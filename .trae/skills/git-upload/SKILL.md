# Git 上传助手

该技能用于帮助用户自动完成 Git 代码提交流程（包括 `add`、`commit` 和 `push` 操作）。

## 适用场景

当用户提出以下要求时，应当调用本技能：
- "帮我把代码上传到 git"
- "提交当前更改"
- "推送到远程仓库"
- "保存并 push 代码"

## 执行步骤

1. **检查状态**：执行 `git status` 确认当前有哪些文件发生了修改或未被跟踪。
2. **生成 Commit Message**：如果用户没有明确提供提交信息，可以简要通过 `git diff` 分析变更内容，生成一个合理的默认 commit message（可询问用户是否满意）。
3. **添加文件**：执行 `git add .` 或者仅添加用户指定的某些文件。
4. **提交代码**：执行 `git commit -m "<生成的或用户提供的提交信息>"`。
5. **推送远程**：执行 `git push`。
6. **异常处理**：若遇到合并冲突、权限问题或没有上游分支等情况，及时向用户反馈并提供解决建议（如使用 `git push --set-upstream origin <branch>`）。
